zakk. Setup Guide

Follow these 4 simple steps to get your website live and ready to manage.

1

Create your free Cloudinary account (for storing photos)

  1. Go to cloudinary.com and click Sign Up Free
  2. Sign up with your email or Google account — no credit card needed
  3. After logging in, you'll see your Cloud Name on the dashboard. Copy it.
  4. Go to Settings (gear icon) → Upload tab
  5. Scroll to Upload Presets and click Add Upload Preset
  6. Give it a name like zakk_preset, set Signing Mode to Unsigned, then Save
  7. Copy the Preset Name (e.g. zakk_preset)
📝 Open the file named config.js
Find the section that says cloudinary: and paste your Cloud Name and Upload Preset:

cloudName: "YOUR_CLOUD_NAME"
uploadPreset: "YOUR_PRESET_NAME"
2

Create your free Firebase account (for storing data)

  1. Go to firebase.google.com and click Get Started
  2. Click Create a Project
  3. Name it zakk-events and click Continue
  4. Disable Google Analytics (optional) and click Create Project
  5. Once created, click Build → Firestore Database → Create Database
  6. Choose Start in test mode → Next → Create
  7. Go to the gear icon ⚙️ → Project Settings
  8. Scroll to Your apps → Click the Web button (</>)
  9. Register your app (nickname: zakk-events) → click Register
  10. You'll see a config object. Copy all 6 values shown
📝 Open config.js again
Find the section that says firebase: and paste all 6 values:

apiKey: "your-api-key"
authDomain: "your-project.firebaseapp.com"
projectId: "your-project-id"
storageBucket: "your-project.appspot.com"
messagingSenderId: "123456789"
appId: "1:123:web:abc123"
✔️ Don't worry — all these values are safe to share. They just let your website read/write data.
3

Deploy your website to the internet

Now we'll upload everything to Vercel so it's live on the web.

  1. Open OpenCode or your terminal / command prompt
  2. Type this command and press Enter:
npx vercel --prod

Wait about 30 seconds. When done, you'll see a URL like https://zakkevents.vercel.app

4

🎉 Start using your admin panel!

Your website is now live and fully self-manageable.

  • Main website: https://zakkevents.vercel.app
  • Admin panel: https://zakkevents.vercel.app/admin.html
  • Password: zakk2025

In the admin panel you can:

  • 📸 Upload photos — they appear on your website gallery instantly
  • 🎨 Change hero photo — the big background image at the top
  • ℹ️ Edit business info — phone, address, taglines
  • 💬 Manage reviews — add customer testimonials
  • 🛎️ Manage services — turn services on/off
🔒 Need to get back to the admin panel? Press Ctrl + Shift + A anywhere on your website, or click the tiny 🔒 Admin link at the bottom-right of the footer.
